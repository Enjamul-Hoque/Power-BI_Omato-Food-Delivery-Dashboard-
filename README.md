# Omato-Food-Delivery-Dashboard-

## Business Problem

The company wanted to track food orders, customer behavior, delivery status, and payment methods in one dashboard. By analyzing this data, the business can identify popular food items, monitor transaction trends, improve delivery performance, and make better business decisions
## Tools Used

* Power BI
* Power Query
* Excel

## DAX Measures Used

* Transaction = COUNTROWS('Sales Data')
* Total Quantity = SUM('Sales Data'[quantity])
* Avg_Quantity = AVERAGE('Sales Data'[quantity])


## KPIs

* Transaction = 15K
* Total Quantity = 2.75K
* Avg_Quantity = 5.0


## Key Insights

* Samosa recorded the highest quantity sold (2,427 units).
* Butter Chicken was the second most ordered item (2,044 units).
* Chole Bhature and Masala Dosa also showed strong customer demand.
* These items are the main contributors to overall order volume.

## Recommendations

* Increase promotions for Samosa and Butter Chicken.
* Ensure sufficient inventory for high-demand items.
* Create combo offers using top-selling foods.
